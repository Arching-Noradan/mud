=========================
BRus MUD Engine readme
=========================

--------------------
Инструкция по сборке
--------------------

Переработано и дополнено с инструкции за авторством Кродо, доступной по адресу http://www.mud.ru/?coder

Зависимости
==================

Для сборки мада вам как минимум понадобятся:

* Современный C++ компилятор
* boost >=1.42.0
* Python >=2.6
* Zlib
* Mercurial (http://mercurial.selenic.com/)

Сборка под студией
==================

1. Устанавливаем boost, кроме заголовочных файлов понадобится библиотека boost_python, убеждаемся, что буст виден студией.
2. Клонируем репозиторий (предполагается, что меркуриал уже установлен)::

    hg clone https://bitbucket.org/Posvist/mud/
3. Создаем чистый проект, идем в Project->Add existing Item и добавляем все файлы из mud\\src\\.
4. В Project->Properties->General выставляем Output Directory в mud\\bin.
5. После чего жмем Build->Build Solution и все должно собраться. 
6. Копируем минимальный набор зон для запуска сервера (lib\\world.template -> lib\\world).
7. Идем в bin, запускаем circle.exe и радуемся жизни.

Cygwin
==================

Этот вариант более мутный, зато приближенный к реалиям на сервере.

1. Качаем и запускаем http://www.cygwin.com/setup.exe
2. Добираемся до списка доступных пакетов, идем в категорию Devel и там отмечаем:
    * boost-devel
    * mercurial
    * gcc
    * make
    * Python 2.6
    * А также все, что там было отмечено по умолчанию и отметилось при выборе этих пакетов

    Качаем, ставим. 
3. 
    Запускаем cygwin, далее в командной строке пишем дословно::

        hg clone https://bitbucket.org/Posvist/mud/

    После чего у вас должна создаться директория mud в которой будет последний код (mud/src). Для обновления кода в будущем всего этого повторять не надо, достаточно будет запустить cygwin, зайти в эту директорию mud ("``cd mud``") и набрать "``hg pull -u``". Физически все это будет лежать в ..\\cygwin\\home\\ваше-имя\\mud\\
4. Находясь в cygwin'е заходим в src (что-то вроде "``cd mud/src``"), пишем "``make test``" и радуемся, если все собирается без приключений. 
5. Теперь нужно скопировать минимальный набор зон для запуска сервера:: 

    cd ..
    cp -r lib/world.template lib/world
6. И наконец - "``./autorun``" - это оно и есть, 4000 порт ждет вас.
